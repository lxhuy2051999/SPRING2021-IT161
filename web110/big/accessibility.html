Accessibility

Web accessibility is one of the most vital aspects that ensures a website to be shown in the first page in any search engine.

Web accessibility is used to describe how well a website is designed and coded so that people with disability can use and interact with that website. “Nowadays, 1/5 of the population of the Earth has a disability and struggle to have access to the same services and products as people without disabilities have”- (Popescu - What accessibility is and why it's so important - uxdesign.cc). After spending a lot of time and effort, developers have created standards and methods to improve websites. As a result, many barriers have been removed and websites become more understandable.

A website is considered accessible when it meets the four principles, known as POUR: Perceivable, Operable, Understandable, and Robust (Everett How & Why Accessibility Matters for SEO searchenginejournal.com). By practicing the POUR principles, we can make the website easier to navigate, find the information, and increase the user experience. There are three levels to measure how well we practice the POUR principles:
A: The most basic accessibility features
AA: The most common barriers for disabled people
AAA: The highest level of accessibility

Search engine optimization (SEO) is a term to describe the process that increases the quality of search results by measuring the numbers of websites with different aspects. When a user gives the search engine a search queries, it finds all websites that matched with the queries. The search engine continues to measure different aspects of each website, websites that get the highest scores after the measurement will be shown on the first pages.

To make our websites gain more views, we must improve the accessibility. A web with good accessibility allows its users to understand the content, use its tools, and interact with it comfortably under different situations. For example, the website can provide captions for a video so people who cannot use speakers or people with impairment can read and understand the content of that video.

In conclusion, web accessibility plays an essential rule to make websites reaches to the viewers
